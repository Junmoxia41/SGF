SGF - MODO ANTI-ESET / ANTI-ANTIVIRUS

Problema detectado:
- ESET esta bloqueando esbuild.exe dentro de node_modules.
- Ese archivo viene del frontend (Vite/esbuild) y provoca que npm install falle.

Esta version del proyecto evita eso asi:

1. El CLIENTE ya viene COMPILADO en:
   client\dist\

2. NO hace falta npm install en client\

3. El SERVIDOR instala SOLO dependencias de produccion:
   npm install --omit=dev --no-audit --no-fund

4. El servidor ahora sirve tambien el frontend compilado.
   URL final:
   http://localhost:3000

COMO USARLO:

1. Ejecuta:
   instalar-todo.bat

2. Luego ejecuta:
   start-servidor.bat

3. Abre en el navegador:
   http://localhost:3000

NO USES en esta PC con ESET:
- npm install dentro de client\
- npm run build del client
- diagnosticos viejos que instalen todo el frontend

Si ESET sigue molestando:
- agrega exclusion a la carpeta del proyecto
- o excluye esbuild.exe si aparece en cuarentena

Ruta tipica detectada por ESET:
- client\node_modules\@esbuild\win32-x64\esbuild.exe

En esta version ya no deberia ser necesario instalar eso en la PC final.
